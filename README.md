# READ_ME - css_button_animation-2
## Description
Ce projet est une variante du projet css_button_animation-1 :
* variation de couleur
* Rectangles transformés en trait
* Taille des traits différents

## SETUP 
```bash
export FLASK_APP=main.py
flask run
```
http://127.0.0.1:5000
